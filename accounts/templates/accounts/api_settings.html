Biometric Punch API

Map each employee's Biometric ID in Add/Edit Employee. Your device should POST punch events to the endpoint below. 

{
  "biometric_id": "1001",
  "punch_type": "in",
  "punch_time": "2026-07-03T09:15:00"
}

punch_type: in or out. punch_time is optional (defaults to now).
